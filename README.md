# My dotfiles

## Usage

- setup WSL
```shell
make build-wsl # must be run as user (not root)
```

- build emacs
```shell
make build-emacs # must be run as user (not root)
```

- deploy setting files
```shell
./install.sh
```

- enable custom keyboard mapping
```shell
./install_xkbmap.sh
```

- enable custom ubuntu shortcuts
```shell
./install_ubuntu_shortcuts.sh
```
